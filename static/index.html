About BLACKBOX NODE

          
Emergency-first Meshtastic AI companion

        

        
      

      

        

        

          
What It Is

          

            
BLACKBOX NODE is a local-first assistant connected to Meshtastic radios for degraded, remote, or emergency conditions.

            
It is built to stay useful when normal internet-first tools are unavailable, slow, or irrelevant.

          

        

        

          
How To Use It

          

            
Use Local Chat to test the offline model directly on this machine.

            
Use the radio side to answer Meshtastic users with compact, packet-safe replies that fit the channel.

            
Use Manage near the model selector to install, switch, or remove compatible GGUF models.

          

        

        

          
Manifest

          

            
This system is designed for resilience over polish.

            
It should remain readable, local, repairable, and useful under stress.

            
The goal is simple: give people a small offline AI that can still help over Meshtastic when infrastructure is weak, damaged, or gone.
